# 🩺 Chronic Kidney Disease Prediction using Machine Learning

## 📌 Project Overview

Chronic Kidney Disease (CKD) is a serious health condition that affects kidney function. Early prediction can help in timely treatment and better healthcare decisions.

This project uses a **Machine Learning approach** to predict whether a patient is likely to have Chronic Kidney Disease based on medical parameters.

A **Random Forest Classifier** is used to train the model and predict CKD outcomes.

## 🔄 Machine Learning Workflow

1. **Data Collection**
   - CKD patient dataset is used for training.

2. **Data Preprocessing**
   - Handling missing values
   - Preparing input features
   - Converting data into suitable format

3. **Model Training**
   - Random Forest Classifier is trained using patient health data.

4. **Model Evaluation**
   - The model performance is evaluated using accuracy.

5. **Prediction**
   - User inputs patient details through the Streamlit interface.
   - The trained model predicts CKD risk.
